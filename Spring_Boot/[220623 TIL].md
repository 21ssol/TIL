## :warning:오늘의 에러

* 블로그형 글쓰기에서 그림 엑박 : 파일 업로드를 images가 아닌 `files`에서 하기

* `invalid bound statement (not found)` : .jsp파일에서 .js파일을 연결해줄 때 경로설정 맨 앞에 / 가 빠짐(static으로 선언되어서 필요함)

  * <script type="text/javascript" src = "/js/cart.js"></script>
