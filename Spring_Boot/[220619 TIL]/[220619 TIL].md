## :warning:오늘의 에러

* `Unknown column in 'where clause` : Sql에서 varchar 타입으로 생성하였기 때문에 ' '로 감싸줘야 한다.
  * `fetch(url="http://localhost:8000/loginCk?id='"+id+"'&passwd='"+pw+"'")`  이렇게 해결

* String id : 스트링타입으로 id를 못 읽어올 때 : Map형식으로 바꿔서 해결
